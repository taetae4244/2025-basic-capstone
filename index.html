what is web ?

웹(Web)은 인터넷을 통해 정보를 주고받을 수 있는 시스템이야.
우리가 흔히 사용하는 웹사이트(네이버, 구글, 유튜브 등)도 웹의 일부
✅ 웹의 기본 개념
웹은 크게 **클라이언트(사용자)와 서버(정보 제공자)**로 나뉨

1️⃣ 클라이언트(Client) - 요청을 보내는 쪽
👉 우리가 사용하는 웹 브라우저(Chrome, Edge, Safari 등)
👉 사용자가 웹사이트 주소를 입력하면 서버에 **요청(Request)**을 보냄

2️⃣ 서버(Server) - 요청을 처리하는 쪽
👉 요청을 받은 후, 웹사이트 데이터를 클라이언트에게 응답(Response)
👉 웹사이트의 HTML, CSS, JavaScript 파일을 보내줌

📌 예시

네이버를 브라우저에 입력
클라이언트(브라우저)가 네이버 서버에 요청 보냄
서버가 HTML, CSS, JavaScript 파일을 클라이언트에게 전달
브라우저가 웹페이지를 화면에 표시

✅ 웹의 구성 요소
웹은 여러 기술로 이루어져 있어!

🔹 HTML → 웹페이지 구조 만들기
🔹 CSS → 웹페이지 스타일 지정 (색상, 크기, 레이아웃)
🔹 JavaScript → 웹페이지에 기능 추가 (버튼 클릭, 애니메이션 등)
🔹 HTTP/HTTPS → 클라이언트와 서버 간 데이터를 주고받는 규칙

✅ 웹사이트와 웹페이지의 차이
💡 웹사이트: 여러 개의 웹페이지가 모인 것! (ex. 네이버, 구글, 유튜브)
💡 웹페이지: 하나의 문서 (ex. 네이버 뉴스, 유튜브 동영상 페이지)

✅ 웹의 작동 방식 한눈에 보기
1️⃣ 사용자가 웹사이트 주소 입력
2️⃣ 클라이언트(브라우저)가 서버에 요청
3️⃣ 서버가 HTML, CSS, JS 파일을 응답
4️⃣ 브라우저가 화면에 웹페이지 표시
